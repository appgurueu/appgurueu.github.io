Schulnotation

Auf Schule:

  • Richtungsvektoren mit einem Pfeil versehen und als Spaltenvektoren notieren
  • Punktvektoren mit Großbuchstaben bezeichnen (A,B,C,...A, B, C, ...A,B,C,...) und als Zeilenvektoren notieren, wobei die Koordinaten von Schrägstrichen getrennt werden
  • Skalare mit Kleinbuchstaben (meist xxx) und Index (x1,x2,x3x_1, x_2, x_3x1​,x2​,x3​)
  • Benannte "Formengleichungen" mit Doppelpunkt und Gleichung mit xxx. Gleichungen müssen direkt eingesetzt werden, E=gE = gE=g ist falsch.
    • Ebenen: E:x=...E: x = ...E:x=...
    • Geraden: g:x=...g: x = ...g:x=...

Terminologie

Komponenten = Einträge.

Punkte

Einfach Vektoren. Werden häufig als Zeilenvektoren notiert: (x/y/z)(x/y/z)(x/y/z).

Richtungsvektoren

Richtungsvektor von Punkt AAA nach BBB: AAA von BBB abziehen: AB‾=B−A\overline{AB} = B - AAB=B−A.

Kolinearität

Zwei Richtungsvektoren a,ba, ba,b sind genau dann kolinear zueinander, wenn die Gleichung xa=bxa = bxa=b mit xxx Skalar erfüllt ist (wenn also ein Vektor nur eine Skalierung des Anderen ist). Dann sind aaa und bbb auch parallel zueinander: a∥ba \parallel ba∥b.

Vektoraddition

Verschiebt einen Vektor um einen Anderen, indem jeweils die Komponenten addiert werden.

Skalare Multiplikation

Skaliert einen Vektor mit einem Skalar, indem jede Komponente mit dem Skalar multipliziert wird. Analog lässt sich ein Vektor auch durch einen Skalar teilen.

Länge

Die Länge eines Vektors lässt sich mithilfe des Satzes des Pythagoras berechnen: Alle Komponenten müssen quadriert werden und anschließend summiert werden. Schließlich muss die Wurzel gezogen werden: ∣(x,y,z)∣=x2+y2+z2|(x, y, z)| = \sqrt{x^2 + y^2 + z^2}∣(x,y,z)∣=x2+y2+z2​.

Normieren

Man normiert einen Vektor, indem man ihn durch seine Länge teilt. Der resultierende Vektor hat immer Länge 111 (Einzige Ausnahme: Der Nullvektor).

an=a∣a∣a_n = \frac{a}{|a|}an​=∣a∣a​

Abstand

Der Abstand zweier (Punkt-)Vektoren ist einfach die Länge des Richtungsvektors: d=∣B−A∣=∣A−B∣d = |B - A| = |A - B|d=∣B−A∣=∣A−B∣. Ob man AAA von BBB abzieht oder andersherum ist wegen der Quadrate bei der [Länge]nbestimmung egal.

Punktprodukt

Addition der Produkte der jeweiligen Komponenten. Ergebnis ist ein Skalar, weswegen das Punktprodukt auch als Skalarprodukt bekannt ist. Gleich dem Produkt der Längen mal dem Cosinus des Winkels α\alphaα zwischen den Vektoren a,ba, ba,b:

Orthogonalität

Orthogonalität heißt, dass der Winkel zwischen den zwei Richtungsvektoren 90°90°90° beträgt. Dann ist der Cosinus und somit das Punktprodukt 000.

a⊥b⇔a⋅b=0a \perp b \lrArr a \cdot b = 0a⊥b⇔a⋅b=0

Parallelität

Bei Parallelität der Richtungsvektoren beträgt der Winkel 0°0°0°, der Cosinus nimmt einen Wert von 111 and und somit ist das Punktprodukt das Produkt der Längen der beiden Vektoren.

a∥b⇔a⋅b=∣a∣∣b∣a \parallel b \lrArr a \cdot b = |a||b|a∥b⇔a⋅b=∣a∣∣b∣. Allerdings prüft man Parallelität am Besten über [Kolinearität].

Für aaa, bbb Richtungsvektoren lässt sich aus a⊥c⇔a⋅c=0a \perp c \lrArr a \cdot c = 0a⊥c⇔a⋅c=0 und b⊥c⇔b⋅c=0b \perp c \lrArr b \cdot c = 0b⊥c⇔b⋅c=0 folgendes herleiten:

Geraden

Formen

Eindeutig gegeben durch zwei verschiedene Punkte A,BA, BA,B. Der Richtungsvektor lässt sich dann wahlweise von AAA nach BBB oder BBB nach AAA bestimmen. Sowohl AAA als auch BBB können als Stützvektor verwendet werden.

Stützvektor (= beliebiger Punkt auf Gerade) plus Skalar sss mal Richtungsvektor bbb der Geraden: g:x=A+sbg: x = A + sbg:x=A+sb.

Schnitte

Einfach Gleichungen gleichsetzen: Punkt-Gerade-Schnitt (Probe, ob Punkt enthalten), Gerade-Gerade-Schnitt, Punkt-Ebene-Schnitt. Fallunterscheidung im Dreidimensionalen.

  • Punkt-Gerade-Schnitt
    • Eine Lösung / Gleichung erfüllt: Punkt liegt auf Gerade
    • Keine Lösung / Gleichung nicht erfüllt: Punkt liegt nicht auf Gerade
  • Punkt-Ebene-Schnitt, analog zum Punkt-Gerade-Schnitt:
    • Eine Lösung / Gleichung erfüllt: Punkt liegt auf Ebene
    • Keine Lösung / Gleichung nicht erfüllt: Punkt liegt nicht auf Ebene
  • Gerade-Gerade-Schnitt
    • Unendlich viele Lösungen: "Gleiche" Gerade, d.h. parallel und kein Abstand zwischen den Geraden
    • Eine Lösung: Schnittpunkt
    • Keine Lösung:
      • Windschief
      • Parallel, wenn Richtungsvektoren kolinear

Ebenen

Eindeutig gegeben durch drei verschiedene Punkte. Parameterform konstruierbar, indem man einen Punkt als Stützvektor wählt und von diesem aus die beiden Spannvektoren bestimmt:

Gegeben Punkte A,B,CA, B, CA,B,C, wähle AAA als Stützvektor. Dann ergeben sich Spannvektoren b,cb, cb,c als b=B−Ab = B - Ab=B−A, c=C−Ac = C - Ac=C−A. Somit ist die Parameterform: E:x=A+sb+tcE: x = A + sb + tcE:x=A+sb+tc mit s,ts, ts,t Skalaren.

Normalenform

Punktprodukt aus Punkt xxx minus Stützvektor PPP mit Normalenvektor nnn muss null sein (siehe [Orthogonalität]): E:n⋅(x−P)=0E: n \cdot (x - P) = 0E:n⋅(x−P)=0. Lässt sich einfach umstellen:

E:n⋅(x−P)=0⇔n⋅x−n⋅P=0⇔n⋅x=n⋅PE: n \cdot (x - P) = 0 \lrArr n \cdot x - n \cdot P = 0 \lrArr n \cdot x = n \cdot PE:n⋅(x−P)=0⇔n⋅x−n⋅P=0⇔n⋅x=n⋅P, wobei n⋅Pn \cdot Pn⋅P als Skalar ausgerechnet werden kann ("vereinfachte" Normalenform).

Bestimmung aus Parameterform: n=b×cn = b \times cn=b×c (Normalenvektor als Kreuzprodukt der SPannvektoren), P=AP = AP=A (Stützvektor / beliebiger Punkt auf Ebene).

Bestimmung aus Koordinatenform: Trivial. Komponenten des Normalenvektors sind Koeffizienten von x1,x2,x3x_1, x_2, x_3x1​,x2​,x3​. Wenn PPP bestimmt werden muss, lässt sich eine beliebige Lösung für x1,x2,x3x_1, x_2, x_3x1​,x2​,x3​ nehmen (bspw. Spurpunkte).

Koordinatenform

Bestimmung aus Normalenform: Setze P=(x1,x2,x3)P = (x_1, x_2, x_3)P=(x1​,x2​,x3​). Es ergibt sich ax1,bx2,cx3=dax_1, bx_2, cx_3 = dax1​,bx2​,cx3​=d mit a,b,c,da, b, c, da,b,c,d Skalaren. In ddd steckt der Abstand zum Ursprung sowie die Länge des Normalenvektors.

Spurpunkte

Als Spurpunkte bezeichnet man die Schnittpunkte einer Ebene mit den Koordinatenachsen. In der Koordinatenform sind Spurpunkte einfach zu bestimmen, indem man jeweils zwei Variablen auf 000 setzt:

  • Schnittpunkt mit x1x_1x1​-Achse: x2=0x_2 = 0x2​=0 und x3=0x_3 = 0x3​=0 setzen
  • Schnittpunkt mit x2x_2x2​-Achse: x1=0x_1 = 0x1​=0 und x3=0x_3 = 0x3​=0 setzen
  • Schnittpunkt mit x3x_3x3​-Achse: x1=0x_1 = 0x1​=0 und x2=0x_2 = 0x2​=0 setzen

Es kann sein, dass nicht alle Spurpunkte existieren (Koeffizienten können 000 sein, dann schneidet sich die Ebene nicht mit der Koordinatenachse):

  • Ein Koeffizient 000 (bspw. x1+x2=1x_1 + x_2 = 1x1​+x2​=1): Schnitte mit zwei Koordinatenachsen
  • Zwei Koeffizienten 000: Schnitt mit einer Koordinatenachse (Normalenvektor ist parallel zu einer Koordinatenachse)

Wenn außerdem d=0d = 0d=0 ist, geht die Ebene durch den Ursprung. In diesem Fall sind unendlich viele Spurpunkte auf allen Achsen, die Koeffizient 000 haben möglich; bei der Spurpunktebestimmung erhält man dann Gleichungen 0=00 = 00=0.

Schnitte

Einfach gleichsetzen: Punkt-Ebene-Schnitt (Probe, ob Punkt enthalten), Gerade-Ebene-Schnitt, Ebene-Ebene-Schnitt.

  • Gerade-Ebene-Schnitt
    • Unendlich viele Lösungen: Gerade liegt in Ebene, d.h. orthogonal zum Normalenvektor
    • Eine Lösung: Schnittpunkt. Wenn zusätzlich der Richtungsvektor parallel zum Normalenvektor ist, steht die Gerade orthogonal zur Ebene.
    • Keine Lösung: Gerade ist parallel zur Ebene, d.h. orthogonal zum Normalenvektor
  • Ebene-Ebene-Schnitt
    • Unendlich viele Lösungen:
      • Auf einer Schnittgerade
      • Gleiche Ebenen: Ebenen parallel (Normalenvektoren kolinear)
    • Keine Lösung: Parallel (Normalenvektoren kolinear)
